GPS 인증 스탬프 투어 웹앱 프로젝트 - 플래깅
송암학생작품대전 작품부문 - 우수상
· 개발기간 : 2022. 05. 26 ~ 2022. 06. 21
· 적용기술 : REST API, HTML5, CSS3, JavaScript, MySQL ···
· 담당직무 : 기획 및 설계, UI/UX 디자인, 웹 퍼블리싱, 동영상 편집
 구체적인 GPS API 사용법을 배우고, 기기 카메라 연동 기능을 연계해 관광을 통한 지역 상권 부흥을 목적으로 한 새로운 개념의 스탬프 투어 웹앱을 기획했습니다. 사용자가 직접 여행지의 숙박, 음식점, 관광지 등 여행 상품으로 코스를 지정한 뒤 저장을 하고, 하루 한 번 진행 가능한 '챌린지'를 시작하면 코스 주변에 도달할 시 QR인증 버튼이 활성화 되어 코드 촬영을 통해 인증할 수 있도록 서비스를 구성했습니다. 한 코스 당 한 번, 랜덤 액수 마일리지 뽑기를 진행할 수 있고 추후 적립된 마일리지는 지역 경제와 관련된 상품권 혹은 교환권 등으로 교환 지급할 수 있도록 기획했습니다.

 실제 구현은 SNS 로그인, 코스 지정 및 저장, 카카오 지도 API 연동, 챌린지 시스템, 마일리지 뽑기 및 적립, 적립 내역 조회까지로 교환 시스템 이외 거의 모든 프론트 및 백이 완성도 있게 제작돼 교수님과 산학 협력 업체 대표님들께 조금만 손을 보면 바로 상업적 이용이 가능한 기획과 구현 수준이라는 평가를 받았습니다.
