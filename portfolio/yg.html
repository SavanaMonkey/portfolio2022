영진직업전문학교 리디자인 및 SNS 로그인 연동 프로젝트
팀장
· 개발기간 : 2022. 03. 16 ~ 2022. 04. 13
· 적용기술 : REST API, Open API, jQuery, MySQL, HTML5, CSS3, JavaScript ···
· 담당직무 : 기획 및 설계(100%), UI/UX 설계(100%), Open API 연구(50%), 보고서 작성(100%)
  코딩을 배우고 처음 돌입한 프로젝트로 프론트엔드 영역까지는 숙련도 대비 짧은 기간이 주어졌기에 온전히 완성할 수 있는 것에 초점을 맞추어 보다 깔끔하게 기획했습니다. 기존에 완벽하지 못해 사용감이 떨어지던 반응형 웹을 완벽히 구현하는 것이 가장 큰 목표였고 부가적으로 다소 딱딱하고 사무적인 분위기와 내용이 많아 직관적이지 못했던 UI를 '직업학교 홈페이지'를 찾는 유저 연령대를 고려하여 보다 젊고 효율적인 분위기로 재구성 했습니다. 목표 설정 이후에는 팀원 각자가 기존 홈페이지를 충분히 탐색한 후 의견을 나누며 카드소팅 기법을 통해 GNB를 정돈했고 발견된 UI/UX의 문제점을 종합해 적절한 해결 방안을 도출했습니다.

  실질적인 구현 단계에선 기획 담당이 스토리 보드를 작성하여 디자인 담당에게 넘기면 한 페이지를 작업하는 동안 프론트엔드 담당은 스토리 보드를 참고하여 필요한 소스를 미리 찾고 연구하는 등 최대한 업무 계획을 겹쳐 시간을 절약하기 위해 노력했습니다. 프론트엔드 작업이 끝난 후에는 피드백과 사용성 검증을 통해 수정 사항을 도출하고 담당자를 배치하여 수정/보완까지 완료했습니다.

 이후 수료 전, 마지막 프로젝트로 기존에 제작해두었던 페이지에 OPEN API를 활용하여 SNS 로그인 연동까지 마친 뒤 프로젝트를 최종 마무리지었습니다. 단순히 정보를 받아와 로그인 할 뿐만 아니라 데이터 베이스까지 활용해 백엔드 영역도 어느정도 구현했습니다.
